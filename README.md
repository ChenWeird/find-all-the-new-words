## 使用

- 把文章放在 ./article 文件夹中
- build 模式用来建立自己的词汇库，即 old.txt，同时也可以将新单词列出来，建议建立的时候挑选较短的文章，一篇一篇来
- 词库建立好了就可以使用 find 模式批量查找生词

## exe文件

- exe文件 release 里面
- 解压后 ，快捷方式需要你自己重新创建
- 如果还需要帮助，你可以参考上面给的示例
